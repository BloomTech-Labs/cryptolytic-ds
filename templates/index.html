Cryptolytic DS API

Routes

  • /trade

    Description: Returns future trade predictions for the given exchange and trading pair

    Input Example: {"exchange_id":"binance", "trading_pair":"eth_btc", "period" : 300}

    Return Example: [{"exchange":"binance","trading_pair":"eth_btc","timestamp":123,"close_pred":1234},{"exchange":"binance","trading_pair":"eth_btc","timestamp":124,"close_pred":1236},{"exchange":"binance","trading_pair":"eth_btc","timestamp":125,"close_pred":1237},{"exchange":"binance","trading_pair":"eth_btc","timestamp":126,"close_pred":1239},{"exchange":"binance","trading_pair":"eth_btc","timestamp":127,"close_pred":1233}]

  • /data/

    Description: Serve static file from data folder.

    Input Example: data/api_info.json
